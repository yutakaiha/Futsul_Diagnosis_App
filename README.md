# アプリ名
- Futsul_Diagnosis_App(フットサル診断アプリ)

# 特徴
質問は３項目に分かれており内応は下記の通り
- 合コンの場での振る舞いに関する質問（10問）
- フィジカル面に関する質問（4問）
- 性格、実スキル面での質問（14問）

上記の合計28問の質問に答えるだけで、フットサルにおいての、
あなたの適性ポジションの診断が可能です。

# 回答方法
シンプルで簡単です。以下の通り

自身の状況に応じて「1, 2, 3」（半角数字）のいずれかの番号を入力するだけ！！
「Yes」→１、「No」→２、「どちらでもない」→３

### 留意事項
※少なくとも一回は回答番号「１」の入力がなければ、正確な診断ができません！！
